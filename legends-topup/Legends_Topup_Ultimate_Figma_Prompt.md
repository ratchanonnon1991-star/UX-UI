# Legends Topup — Ultimate Figma AI Prompt (Starter Edition)

> **Brand:** Legends Topup  
> **Tagline:** Top Up Faster. Play Better.

## Objective
Design a world-class universal game top-up platform supporting all major games. The experience should feel like a blend of Steam, Epic Games Store, Riot Client, Codashop, UniPin, Amazon, and Shopee while maintaining a unique premium gaming identity.

## Design Principles
- Mobile-first
- Premium Gaming UI
- Dark Theme
- Glassmorphism
- Neon gradients (Purple, Blue, Cyan, Gold)
- Rounded cards (16–24px)
- Responsive with Auto Layout
- WCAG AA contrast
- Production-ready Figma components
- 8pt spacing system

## Brand Personality
Fast • Secure • Premium • Friendly • Global • Fun • Trustworthy
