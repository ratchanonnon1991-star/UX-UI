# Illustration & Visual Style

## Style
Original premium gaming universe. Anime-inspired, fantasy + sci-fi, friendly and polished.

## Elements
- Cute mascot guide
- Floating diamonds
- Coins
- Treasure chests
- Magic portals
- Energy particles
- Soft neon glow
- Dark navy background
- Blue, purple, cyan, gold accents

## Mascot Usage
- Hero: welcoming users
- Loading: delivering crystals
- Success: celebrating
- Error: searching with magnifier
- Empty state: holding treasure box

## Do Not Use
- Copyrighted game characters
- Direct copies of game art
- Overly busy backgrounds
- Low contrast images behind text
